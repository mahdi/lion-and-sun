# OpenSpec Instructions

Use OpenSpec for this project.

- Read `openspec/project.md` before planning or implementing changes.
- For feature work, create or update a change under `openspec/changes/`.
- Keep requirements in `specs/*/spec.md` files and implementation detail in `design.md` and `tasks.md`.
- Validate changes with `openspec validate <change-name> --strict` before considering work complete.
