# Bootstrap Watch App

## Why

The repository is empty and needs an initial watchOS app, emblem asset, and complication support.

## What Changes

- Add a standalone watchOS 10+ SwiftUI app named `Lion and Sun`.
- Display the Lion and Sun emblem with `Javid Shah!` on the app's first screen.
- Add a WidgetKit extension supporting all modern watchOS accessory complication families.
- Store the source emblem artwork locally in the Xcode asset catalogs.

## Capabilities

- watch-app
- complications

## Non-Goals

- iPhone companion UI.
- User settings, data sync, backend services, or dynamic timelines.
- Legacy ClockKit complications.
